# 🗂️ Console-Based Task Manager

## 📌 Project Overview

This project is a Console-Based Task Manager built using JavaScript and integrated into a styled HTML and CSS layout.

The application allows users to add tasks using prompts, store them in an array as objects, and filter completed tasks. While the logic runs in the console, the project is structured within a responsive Kanban-style interface.

---

## 🚀 Features

- Add tasks using `prompt()`
- Store tasks as objects in an array
- Automatically generate unique incremental IDs
- Validate task status (`todo`, `doing`, `done`).
- Limit total tasks to 6.
- Alert user when the board is full.
- Display:
  - All tasks in the console
  - Completed tasks only
- Responsive layout (Desktop, Tablet, Mobile)
